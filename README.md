# AI Resume Job Finder

## Overview

AI Resume Job Finder is a Python-based application that analyzes resumes and recommends matching jobs using live job data.

The application extracts resume content, calculates ATS scores, identifies missing skills, and fetches real-time job opportunities through the Adzuna API.

---

## Features

* Resume Upload (PDF)
* Resume Text Extraction
* ATS Score Calculation
* Skills Detection
* Missing Skills Analysis
* Live Job Search using Adzuna API
* Resume-Job Matching
* Direct Apply Links

---

## Tech Stack

* Python
* Streamlit
* Pandas
* PDFPlumber
* Scikit-Learn
* Sentence Transformers
* Requests API
